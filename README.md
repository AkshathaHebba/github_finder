# GitHub Finder

GitHub Finder is a web application implemented using the Tailwind CSS framework with DaisyUI. It provides a user-friendly interface for searching GitHub repositories and users.

## Installation - UI Design

To install Tailwind CSS, please refer to the official [Tailwind CSS installation guide](https://tailwindcss.com/docs/guides/create-react-app).

For installing DaisyUI, follow the instructions provided in the [DaisyUI installation guide](https://daisyui.com/docs/install/).

# React Application
## GitHub User Repos

GitHub User Repos is a web application that allows users to retrieve repositories of a specific GitHub user using the GitHub API.

### Description

With GitHub User Repos, you can easily search for a GitHub user and retrieve a list of their repositories. The application leverages the GitHub API to fetch the repositories associated with the specified user. This information is then displayed in a user-friendly format, providing details such as repository name, description, and the number of stars and forks.

Whether you are exploring someone's projects or conducting research on a specific user's repositories, GitHub User Repos simplifies the process by providing a streamlined interface to access the desired information.

### How to Use

To use the application, simply enter the username of the GitHub user you wish to retrieve repositories for. GitHub User Repos will then make a request to the GitHub API and fetch the repositories associated with the provided username. The results will be displayed in a clear and organized manner, making it easy to navigate through the user's repositories and explore their respective details.

Start discovering GitHub repositories with GitHub User Repos and delve into the diverse range of projects available on the platform.

### Deployment

The application is deployed on Vercel and can be accessed at [GitHub Finder on Vercel](https://github-finder-gi3mv2oze-akshathahebba.vercel.app/).

Feel free to explore and discover GitHub repositories and users using the GitHub Finder application.
